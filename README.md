新教学 Wiki （2019）的全部源代码，基于 [mkdocs](http://www.mkdocs.org/) 。

本地构建和运行站点方法如下：

```bash
git clone https://github.com/c4pr1c3/cuc-wiki.git

cd cuc-wiki 

pip install mkdocs mkdocs-material pygments

# 使用 mkdocs 内置服务器提供本机可访问网站服务： http://127.0.0.1:8000/
mkdocs serve

```
